# Registration Events

## React + TypeScript + Vite

## Install and Run

Please do next steps:

1. Clone repo

```bash
git clone https://github.com/Vladimir-Maselskiy/event-app-front.git
cd event-app-front
```

2. Install dependencies

npm install

3. Run in dev mode

npm run dev

4. Open in browser

http://localhost:5173
